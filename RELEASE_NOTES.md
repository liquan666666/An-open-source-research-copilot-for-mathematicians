## 🛠️ 技术实现

## 🔧 安装和使用

### 使用 Docker（推荐）

```bash
# 克隆仓库
git clone <repository-url>
cd An-open-source-research-copilot-for-mathematicians

# 配置环境变量（可选）
cp apps/api/.env.example apps/api/.env
# 编辑 .env 添加 AI API 密钥

# 启动所有服务
docker-compose up --build

# 访问应用
# API 文档: http://localhost:8000/docs
# 前端: http://localhost:3000
```

### 本地运行

```bash
# 后端 API
cd apps/api
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
uvicorn server.main:app --host 0.0.0.0 --port 8000 --reload

# 前端（新终端）
cd apps/web
npm install
npm run dev
```

详细步骤请查看 [START.md](./START.md)

## 🎯 升级指南

### 从演示版升级

如果你之前使用的是演示版本：

1. **备份数据**: 导出所有数据（GET /export/json）
2. **更新代码**: `git pull origin claude/fix-search-issue-e91UF`
3. **重建容器**: `docker-compose up --build`
4. **迁移数据**: 使用导入功能恢复数据

### 配置 AI 功能

1. 获取 API 密钥（Anthropic 或 OpenAI）
2. 编辑 `apps/api/.env`
3. 重启服务: `docker-compose restart api`
